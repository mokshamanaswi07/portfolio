Smart In-Car Child Heatstroke Prevention System

Designed a low-cost child safety alarm system using Arduino to detect a child’s presence and monitor car cabin temperature. When unsafe conditions are detected, it triggers alerts through a buzzer, LED, and vehicle alarm, and sends SMS notifications to caregivers via GSM. The system provides an affordable retrofit solution to prevent vehicular heatstroke incidents.

Smart Exam Desk Anti-Cheating Alert System

Developed an 8051-based anti-cheating system using vibration and IR sensors to monitor exam desk activity. The system detects abnormal movements and triggers alerts via buzzer, LED, and LCD display. Implemented using AT89S52 with Assembly programming, ensuring reliable detection with reduced false positives.
